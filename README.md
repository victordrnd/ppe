# Projet PPE

## Commande Git

### Pour ajouter les fichiers modifiés dans un pack

` git add * `

### Pour nommer le pack

` git commit -m "nom du pack" `

### Pour envoyer le pack sur le Git

` git push origin master `

### Pour mettre à jour le dossier depuis Git

` git pull `
